> **Deep Learning based Intrusion Detection System capable of identifying Zero-Day attacks in real-time.**

This project represents a shift from signature-based detection (Alerting on known bad) to **behavior-based detection** (Alerting on anomalies).

### âš¡ Key Capabilities
*   **Real-Time Traffic Ingestion:** Utilizes `Scapy` to capture live packets from network interfaces in promiscuous mode.
*   **ML-Based Anomaly Engine:** Deploys a trained **Random Forest/Isolation Forest** model to classify traffic flows as *Benign*, *DDoS*, *Port Scan*, or *Botnet*.
*   **Automated Response (IPS):** Integrates with `iptables` to **automatically block** source IPs exhibiting malicious behavior, effectively acting as an IPS.
*   **Dashboard Visualization:** Custom Streamlit dashboard for real-time traffic monitoring and alert visualization.
